USER STORY:

“Workout Today”
When clicked:
1) Show day picker of available WORKOUT days in this cycle (future/uncompleted depending on mode).
2) Then show action picker:
   - SWAP REST (swap today’s rest with the selected workout day)
   - PUSH REST (make selected workout day happen today and shift other workouts forward, moving rest later)
Show a preview of before/after order/schedule.

BEHAVIOR DETAILS:

A) Calendar-based cycles:
- SWAP REST:
  - Today’s date becomes the selected workout day
  - The selected workout day’s scheduled date becomes Rest
  - No past dates modified; only this cycle schedule.
- PUSH REST:
  - Selected workout day becomes today
  - All workouts between today and the selected workout day shift earlier by one slot
  - Rest moves to the selected workout day’s original date (or to the end of that block)
  Example:
    Mon Rest, Tue Chest, Wed Back, Thu Shoulders
    Push Rest => Mon Chest, Tue Back, Wed Shoulders, Thu Rest

B) Completion-based cycles:
- Treat “today” as the next position in remaining sequence.
- If next is Rest and user chooses “Workout Today”:
  - SWAP REST: swap Rest with the chosen workout day within remaining uncompleted days.
  - PUSH REST: remove Rest from next position and insert it after the chosen workout day (or after N workout days).
- Do not rewrite already-completed days.
