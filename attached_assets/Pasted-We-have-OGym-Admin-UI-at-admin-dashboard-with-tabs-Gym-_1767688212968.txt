In Gym Requests tab, each request row/card should have:

A View Details button (or clicking the row opens details)

Clicking it opens a Modal / Drawer / Full Page (your choice) showing:

Gym Name

Phone

Address

Point of Contact Name

Point of Contact Email

City

State

Country

Gym Size (0–50, 51–150, 151–300, 300+)

Number of Trainers

Mark request as Approved

Create a record in gyms table (if not already created)

Mark request as Rejected

Store rejection_reason (optional text input in modal)
