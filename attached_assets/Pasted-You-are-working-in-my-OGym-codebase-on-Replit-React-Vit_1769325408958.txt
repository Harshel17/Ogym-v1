USER STORY:
Member has a cycle: Day1 Chest, Day2 Back, Day3 Shoulders, Day4 Rest.
On a given day, the member wants to do a DIFFERENT workout day (e.g., Shoulders) instead of the planned day (e.g., Chest).
When they pick another workout, they must choose how to adjust the plan for THIS CYCLE ONLY:
- SWAP: swap the selected workout day with today’s planned day
  Example: Day1 Chest, Day2 Back, Day3 Shoulders
  Today Day1, select Day3, SWAP => Day1 Shoulders, Day2 Back, Day3 Chest
- PUSH: bring the selected workout day to today and shift the other workout days forward in order
  Example: Day1 Chest, Day2 Back, Day3 Shoulders
  Today Day1, select Day3, PUSH => Day1 Shoulders, Day2 Chest, Day3 Back

IMPORTANT RULES:
- Changes apply ONLY within the current cycle instance (do not affect future cycles or templates).
- Must not change past history; do not rewrite already-completed days.
- Must not break existing Rest/Reschedule/Train Anyway flows.
